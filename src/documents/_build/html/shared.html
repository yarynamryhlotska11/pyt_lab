shared.color_font_processor module¶

Color Processor Module

This module provides functionality for color processing using the colorama library and font options using the pyfiglet library. It initializes colorama, enumerates available colors and fonts, and offers functions to display color and font options.

Functions: - display_colors() -> None:: Displays a numbered list of available color options. - display_fonts() -> None:: Displays a numbered list of available font options.

Usage: 1. Import the module: import shared.color_font_processor 2. Use the provided functions to display color and font options.
