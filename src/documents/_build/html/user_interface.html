user_interface.menu_builder module¶

Module Overview: Menu Implementation

This module introduces an abstract base class named ‘Menu’ that provides the structure for various menu implementations. The ‘Menu’ class is intended to be subclassed, offering a blueprint for creating specific menu functionalities.

Classes: - Menu: An abstract base class defining the structure for different menu implementations.

Usage: This module serves as a foundation for defining menu structures. Subclasses of ‘Menu’ should implement the abstract ‘run’ method to define the specific behavior of each menu when executed.
