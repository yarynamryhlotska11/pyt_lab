user_interface.menu_ui.lab2.calculator module¶

Module for implementing a calculator menu.

class user_interface.menu_ui.lab2.calculator.CalculatorMenu¶

Bases: Menu

The ‘CalculatorMenu’ class offers a menu-driven interface for a calculator application. It utilizes functionalities from the ‘CalculatorService’ class to perform calculations based on user input. The ‘CalculatorMenu’ class inherits from the abstract base class ‘Menu’ and implements a user interface for a calculator system.

static display_menu()¶

Display the calculator menu.

run()¶

Run the calculator program.

Initializes the ‘CalculatorService’ and runs the calculator menu in a loop until the user chooses to exit.
